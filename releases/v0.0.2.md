# Framora 0.0.2

## 中文

- 初始化公开发布仓库。
- 新增 `latest.json`，用于应用内检查是否存在新版本。
- 本版本用于验证从本地 `0.0.1` 检测到远程 `0.0.2` 后显示“更新”按钮。
- 本次不提供 App 内自动下载，点击更新按钮后打开 GitHub Releases 页面。

## English

- Initialize the public release repository.
- Add `latest.json` for in-app update checks.
- This release validates showing the Update button when local `0.0.1` detects remote `0.0.2`.
- This iteration does not provide in-app auto download; the Update button opens the GitHub Releases page.
